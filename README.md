# Password Manager

A secure, command-line password manager built with Python.  
It uses encryption to safely store your passwords in an encrypted vault protected by a master password.

---

## Features

- Encrypted password vault stored locally  
- Master password authentication  
- Add and retrieve passwords for different sites  
- Generate strong random passwords  
- Copy passwords securely to clipboard  
- Simple, easy-to-use command-line interface

---
